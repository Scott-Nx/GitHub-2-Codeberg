# GitHub-2-Codeberg

A tool for migrating repositories from GitHub to Codeberg.

## Features

- Migrate repositories from GitHub to Codeberg
- Preserve commit history, branches, and tags
- Simple command-line interface

## Requirements

- Git
- A GitHub account with repositories to migrate
- A Codeberg account

## Installation

```bash
git clone https://github.com/yourusername/GitHub-2-Codeberg.git
cd GitHub-2-Codeberg
```

## Usage

1. Configure your GitHub and Codeberg credentials
2. Run the migration script
3. Verify your repositories on Codeberg

## Configuration

Create a configuration file or set environment variables:

```bash
export GITHUB_TOKEN="your_github_token"
export CODEBERG_TOKEN="your_codeberg_token"
```
